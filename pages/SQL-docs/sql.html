Sebastian Gatica

Developer-Full-Stack.

Soy un estusiasta, cuando se trata de aprender nuevas habilidades en el ambito tecnologico. Deseo formar parte de un equipo de trabajo, para aportar un gran valor, en la solucion de problemas, optimización de procesos, trabajar utilizando metodologias agiles, asegurando un producto de calidad, comprobando que el proyecto posea una arquitectura web de baja cohesión y altisima escalabilidad.

El stack utilizado por mi para el desarrollo, esta basado en el ecosistema de Javascript, el manejo de una variedad de paquetes atravez de npm, permitiendome hacer desde conexiones a bases de datos de diversas clases, hasta encriptados de contraseñas, generacion de tokens de acceso limitado con fecha de caducidad configurable, similares a los de github, utilizados para realizar operaciones, como por ejemplo clonar repositorios.

React

Desarrollo basado en componentes funcionales, ultilizacion de hooks para el manejo de estados, y Redux para globalizar los valores de los estados. Implementacion de memo(), para optimizacion, y de esa forma evitar la repeticion de , y de formik para manejo de formularios.
